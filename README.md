# AI Interview Platform

A lightweight web application that generates interview questions dynamically by analyzing a candidate's resume text, skills, and role.

## Features

- Resume-aware question generation.
- Skill detection from keywords (JavaScript, React, Python, SQL, DevOps, Testing, System Design, Data Science).
- Difficulty adapts to detected seniority signals.
- Role-specific behavioral fallback prompts.

## Run locally

```bash
npm start
```

Open `http://localhost:3000`.

## Test

```bash
npm test
```
